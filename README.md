# ETL Dashboard

Clone repo, install dependencies and open as jupyter lab notebooks.

```bash 
git clone https://github.com/josephmachado/etl-dashboard.git
cd etl-dashboard
pip install -r requirements.txt
jupyter lab
# open etl-dashboard/main.py
```
